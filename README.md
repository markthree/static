# static

自建的静态文件资源库

<br />

## URL

```shell
https://markthree-static.deno.dev
```

<br />

## Stack

完全由 [Deno](https://deno.land/) 的 [file_server](https://deno.land/std@0.167.0/http/file_server.ts?s=serveDir) 标准库搭建，并部署在 [Deno Deploy](https://deno.com/deploy) 之上。
